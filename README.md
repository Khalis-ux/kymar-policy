# kymar-policy

Public-facing pages for the Kymar iOS app, served via GitHub Pages at
<https://khalis-ux.github.io/kymar-policy/>.

- `privacy-policy/index.md` → privacy policy
- `support/index.md` → support page
- `index.md` → landing page
- `_config.yml` → Jekyll config

Edit, commit, push — Pages rebuilds in ~1-2 minutes.
